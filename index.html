HousePricingEDA - Ciencia de datos

Este proyecto de análisis de datos de casas sigue un enfoque estructurado para procesar y preparar datos para modelos de aprendizaje automático. Comenzamos con un análisis exploratorio de datos (EDA) para entender patrones y detectar anomalías. Luego, realizamos limpieza de datos para manejar valores nulos o atípicos. Convertimos variables categóricas en numéricas utilizando técnicas como codificación one-hot y MultiColumnLabelEncoder, asegurando que los datos sean compatibles con algoritmos de aprendizaje automático. Finalmente, analizamos la correlación entre características para seleccionar las más relevantes, optimizando la precisión de las predicciones.
